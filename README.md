### 🚗 Geospatial Crash Analytics — NYC Motor Vehicle Collisions
> *Borough-aware crash severity classification with explainable AI*

Built a crash severity classifier and analytics dashboard on NYC collision data — with a focus on making the model interpretable and actionable for city planners.

**What I built:**
- **Random Forest classifier** for crash severity by borough — **AUC: 0.90**
- **Polygon joins** to impute 1,000+ missing location records from incident data
- **SHAP-based model interpretability** to explain which factors drive severe crashes
- **Custom resampling** to handle skewed class distribution
- **Interactive dashboard** with crash heatmaps, borough clustering, and root cause analysis tools

**The business question I started with:** *Where do the most severe crashes happen, why, and what can be done about it?*
